# 3D Haunted House (TypeScript)

A Three.js haunted house scene from the **Three.js Journey** course, refactored into TypeScript with a modular, class-based architecture.

## Tech Stack

- Three.js
- TypeScript
- Vite
- GLSL (custom shaders)

## Features

- Modular scene classes (House, Graves, Ghosts, Lights)
- TypeScript strict mode
- Responsive canvas + OrbitControls
- PBR textured floor with AO, normal and displacement maps

## Getting Starteda

```bash
npm install
npm run dev
```
